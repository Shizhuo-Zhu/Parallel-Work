To get all resources:
  cd ./backend // in backend folder
  go run main.go
  open a new terminal
  cd ./frontend // in frontend folder
  npm start

To get resources by id:
  cd ./backend
  go run main.go -id="instance_name"(e.g. go run main.go -id="instance-20250414-214402")
  open a new terminal
  cd ./frontend
  npm start
