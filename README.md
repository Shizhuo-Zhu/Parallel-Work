To get all resources:

    # in backend dir
  
    go run main.go
  
    # open a new terminal
  
    # in frontend dir
  
    npm start

To get resources by id:

    # in backend dir
  
    go run main.go -id="instance_name"(e.g. go run main.go -id="instance-20250414-214402")
  
    # open a new terminal
  
    # in frontend dir
  
    npm start
